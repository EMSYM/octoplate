# Octoplate

Boilerplate for [Octopress](http://octopress.org) based on Twitter Bootstrap 3.

## Install

    $ cd octopress
    $ git clone git://github.com/mjhea0/octoplate.git .themes/octoplate
    $ rake install['octoplate']
    $ rake generate && rake preview
    $ rake deploy
